# Contributor Covenant Code of Conduct

This project adopts version 2.1 of the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

A condensed summary of the rules, written in our own words:

- Be patient. Remember that everyone is volunteering time.
- Be specific. Criticism lands better when it names the code, not the coder.
- Assume good faith. If a message reads as rude, ask for clarification before
  reacting.
- Unwelcome conduct — harassment, discrimination, personal attacks — ends
  participation.

## Scope

These rules apply to every space the project touches: issues, pull requests,
discussions, chats, conference talks, and any channel where a contributor
represents Memee.

## Reporting

If you experience or witness conduct that concerns you, write to
**info@memee.eu**. The maintainers will treat the report confidentially.
Expect a reply within 48 hours.

We will not retaliate against anyone filing a report in good faith. We will
act proportionately — a private conversation for a first small incident, a
temporary ban for a pattern, a permanent ban for a severe violation.
